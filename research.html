What are Bots?

Chat bots are computer programs that mimic conversation with people using artificial intelligence. They can transform the way you interact with the internet from a series of self-initiated tasks to a quasi-conversation.

What can we do with Skype Bot?

How to add a Skype Bot:

  • Android - Select the Recents tab, tap the floating action button, and then tap the bot icon. Alternatively, you can find available bots in your Contacts.
  • iOS -Select the bot icon in the Recent tab. Alternatively, you can find available bots in your Contacts.
  • Mac - Select Contacts, then click Add Bot...
  • Web - Select Discover Bots.
  • Windows Desktop - Select the Add bots icon.
  • Skype Preview for Windows 10 - Select the Bot tab.

Scroll through the list of available bots to find the one you want. When you've selected one, select Add bot (or Get Started).

Some bots will be certified by Skype to show its authenticity, you can normally tell is authentic by because of three things.



  1. The Skype bot profile will say Skype Certified.
  2. Normal avatars are round. A Skype bot's avatar is hexagon-shaped.
  3. You will see a gray tick mark next to the contact name.

You can block or remove a bot the same way you would a regular Skype contact. Get help with blocking or removing a contact. If you want to report feedback on a specific bot, contact the bot developer directly. You'll find the name of the developer in the 'created by' section of the bot's profile.

You can also have group chats with bots for those that have that feature enabled.

Most Skype bots can be found in the Skype Directory however there also also certain bots that cannot be found here and must instead be added via their link - the link just directs you to a webpage which will let you add the specified bot as contact to your account.

Similar apps to our project:

Since using Skype for our bot seemed very promising, we decided to see if there were any bots which would be performing a similar task so we would have a better idea of how to inform our own designs.

To this end I had decided to look at the SkyScanner bot on Skype. SkyScanner does flight and hotel room bookings and as a result would be similar to our gym booking system - both concentrating on booking certain places for the customer at a time that is available. Below you can see how the interaction with the bot unfolded.


When a user first enters the bot interface, they will be met with a welcome message and a few prompts to get them started. Here we have the option to see an example of booking a flight to help the user get to know how to use the system. For first time users they are prompted whether they wish to set certain places as default locations to provide a more smooth user journey for subsequent journeys.



After this, we have then decided to actually book a flight. For this we are prompted for our destination, where we intentionally made a typo in the response to see if the bot could deal with that. In the end it had managed to understand we meant New York and after further prompts for date and starting location as well as other options gave us some options to pick from (presented in the form of cards). It took around 10 seconds for the bot to query our search and return the results and each result had the option to book, see the cheapest options and set a price alert. If you were to click "book" it would then redirect you to the SkyScanner website to complete the bookings from the options shown there. If you were to click "show cheapest options", it would show the cheapest 5 flights in the bot itself in the form of more cards where you had the option to book, again redirecting you if you were to click "book".




Messenger Bot

General info:

Facebook's bot is a chatbot existing within Facebook's messenger app. Facebook's messenger has over 900 million users a month and is one of the biggest chatting platforms available at the moment.

Pros:

Facebook messenger can be downloaded on any mobile devices and is also available on desktops. No compability problems with Android, Microsoft or IOS the user just needs to have the app downloaded on his device or book a class with his laptop.

In order to set up a facebook bot, the client has to have a facebook page. Now, in order for people to find your bot, Facebook has set up many different ways to link users to the bot:

  • Messenger Codes
  • Messenger Links

3 things that can be done using the FB bot:

  • Send/receive API
  • Generic message templates
  • Welcome screen + Null state CTAs

Cons:

Only way to communicate with the bot is with texting (compared to siri which can type).

Need a FB account to access to the page. This is important for the user journey because it means Nuffield Health would need to incorporate the Facebook login in its system.

User Journey

The user opens messenger. Opens a chat with the bot and then starts conversation asking something similar to: "book a gym class for --/--/-- at around --:---" The bot then responds with information about session around that time and sends information back to user within a box with options book or other gym class or cancel. The user selects one of the boxes and until he books a gym class. The conversation with the bot can now end.

Overall, the FB bot seems like a good match for the platform. Note: it would be worth investigating the existing login for Nuffield customers as introducing a Facebook Login might not be possible with the system.

    What is a Digital Assistant?

    A digital assistant, also called a virtual assistant, is an application program that can understand natural language and complete electronic tasks for the end user. Such tasks, which might have been performed by a personal assistant or secretary in previous decades, include taking dictation, reading text or email messages aloud, looking up phone numbers, anticipating requests, placing calls and reminding the end user about appointments. Popular digital assistants currently include Apple's Siri, Google Now and Cortana, the digital assistant built into Windows Phone 8.1. Today's digital assistants are programmed with artificial intelligence, machine learning and voice recognition technology. As the end user interacts with his digital assistant, the AI programming uses sophisticated algorithms to learn from data input and become better at predicting the end user's needs. Tomorrow's digital assistants will be built with more advanced cognitive computing technologies which will allow a digital assistant to understand and carry out multi-step requests and perform more complex tasks such as making a plane reservation.

    Alexa

    Alexa List Skill API

    The List Skill API has a bi-directional interface that ensures lists are updated across all channels. That means the API notifies developers when a customer tells Alexa to add something to their list or makes a change to an existing item. Alexa understands the user’s speech request, converts it to a To-do, and sends you a notification with the new item that was added to the list. The List Skill API also updates the lists for Alexa when users make changes to their lists online or in your mobile application. We could apply this to the gym booking request, user adds book a gym class to his/her To-do list or user adds he/she has to attend the gym on a specific day and then Alexa sends a notification to the user’s phone. On the day of gym class, when user asks Alexa what’s on my To-do list, Alexa would remind him/her of going to the gym today. Alexa List Skill API is available now but in limited participation beta.

    Alexa Skills Kit

    By using Alexa skills, we can engage voice interaction for getting a reminder from Alexa on the day of the gym class. Alexa has a skill which is called Calendar Reader, if we could sync the user calendar with Alexa, we would be able to get updates on the events in our case the gym class everyday, by asking Alexa what events are happening today? Tell me more about this event Alexa. Of course this is only possible when we build a bot which is able to add the gym class to user’s calendar immediately after user has booked the class.

    Account Linking Feature

    This feature, which allows customers to easily link their accounts with an existing account or service. Account linking allows our skill to securely authenticate a user with popular websites and services like Twitter, Facebook, and many others. Once the user has granted your skill access to the external account, the skill can perform operations with that account on behalf of the user. In our use case, user can log in to his/her Facebook, Skype, Slack account to be able to send messages to the chat bot for booking a gym class or being informed on the gym classes available.
